## 📝 Sobre este Trabalho

Este repositório contém o resumo crítico do artigo acima, elaborado como atividade avaliativa da disciplina de **Projeto de Software**.

O resumo aborda os seguintes tópicos extraídos do artigo:

- O problema que motivou a criação da OCL: os limites expressivos dos diagramas UML
- As três características centrais da linguagem: tipada, declarativa e livre de efeitos colaterais
- Os cinco tipos de expressões suportados: invariantes, inicialização, derivação, consultas e contratos
- O sistema de tipos e o mecanismo de navegação pelo modelo
- Ferramentas de validação e verificação, com destaque para o USE
- Os usos da OCL além do UML: transformações de modelos, DSLs e geração de código
- Reflexão crítica sobre a relevância da OCL no desenvolvimento de software moderno
